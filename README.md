# PulseShop

## A fast E-Commerce Shopping App made with React.

### Live @ https://pulseshop.pages.dev/

### Run Locally

#### Server

#### 1. Run "npm install".
#### 2. Run "node start server.js".

#### Client

#### 1. Run "npm install".
#### 2. Run "npm run dev".
#### 3. Start the required port on browser.
